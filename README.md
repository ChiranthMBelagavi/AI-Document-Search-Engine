
# AI Document Search Engine

## Install Ollama

Download:
https://ollama.com

Run model:

ollama run llama3

---

## Backend Setup

python -m venv venv

Windows:
venv\Scripts\activate

cd backend

pip install -r requirements.txt

python app.py

---

## Frontend Setup

cd frontend

npm install

npm run dev

---

Frontend:
http://localhost:5173

Backend:
http://127.0.0.1:5000
